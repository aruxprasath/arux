Intelea'25

Bug Bust (Code Debug)

Date: September 24, 2025

Objective

Improve participants' coding and debugging skills. Sharpen problem-solving and logical thinking ability through error finding and programming challenges. Encourage teamwork, creativity, and learning in a competitive environment.

Event Format

  • Round 1: Error Finding (5 Q in Python + 5 Q in C) — 10 min
  • Round 2: Program will be given as xerox copy. Participants must clear the errors and execute the program on the system to produce the correct output — 15 min

Rules

  • Participation is only for single participants; no groups allowed.
  • All rounds are time-bound; no extra time will be given.
  • Use of mobile phones, calculators, or external help is strictly prohibited.
  • Only the answers submitted within time will be evaluated.
  • There is no negative marking.
  • In case of a tie, a sudden-death coding challenge will decide the winner.
  • Judges' and organizers' decisions will be final.
